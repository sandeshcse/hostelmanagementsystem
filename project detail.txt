# Hostel Management System Documentation

## Project Overview
This is a comprehensive Hostel Management System built using PHP and MySQL. The system allows administrators to manage student hostel accommodations, including room allocation, student registration, and fee management.

## Technologies Used
- **Backend**: PHP
- **Database**: MySQL
- **Frontend**: 
  - HTML5
  - CSS3
  - JavaScript
  - Bootstrap 4.1.0
  - jQuery 3.3.1
- **Additional Libraries**:
  - Chart.js (for data visualization)
  - DataTables (for table management)
  - FullCalendar (for calendar functionality)
  - Morris.js (for charts)
  - Perfect Scrollbar (for custom scrollbars)

## Project Structure
```
HostelManagement-PHP/
├── admin/           # Admin panel files
├── assets/          # Static assets (CSS, JS, images)
├── DATABASE FILE/   # Database schema and data

├── includes/       # Common PHP includes
├── scss/           # SCSS source files
├── student/        # Student portal files
├── gulpfile.js     # Gulp configuration
└── package.json    # Project dependencies
```

## Database Structure
The system uses the following main tables:
1. `admin` - Administrator accounts
2. `adminlog` - Admin login history
3. `courses` - Available courses
4. `registration` - Student hostel registrations
5. `rooms` - Hostel rooms information
6. `states` - List of states
7. `userlog` - Student login history
8. `userregistration` - Student user accounts

## Development Setup
1. Clone the repository
2. Import the database schema from `DATABASE FILE/hostelmsphp.sql`
3. Configure database connection in includes/config.php
4. Install dependencies:
   ```bash
   npm install
   ```
5. Run development server:
   ```bash
   gulp serve
   ```
